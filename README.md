# MouseClick

## 功能概述
- 在窗口中实时显示最近 3 秒内的鼠标点击位置。
- 过期的点击自动消失：3 秒前的记录会被清理。
- 所有历史点以蓝色实心圆显示，最新点击点以红色实心圆显示。

## 运行环境
- 操作系统：Windows 10/11。
- 编译器：MSVC 2022 64bit。
- Qt 版本：Qt 6.10.0（Widgets 模块）。
- IDE：Qt Creator 18.0.0（使用默认 CMake 工程即可）。

## 构建步骤（Qt Creator）
1. 打开 Qt Creator，选择 **File -> Open File or Project...**，加载本仓库中的 `CMakeLists.txt`。
2. 在 Kits 中选择 *MSVC 2022 64bit* 与 Qt 6.10.0 的工具链，点击“Configure Project”。
3. 进入项目后，直接点击左下角绿色“播放”按钮进行构建与运行，程序会以 Release/Debug 方式启动（可在构建配置中切换）。

## 使用说明
1. 程序启动后会出现一个空白窗口。
2. 在窗口任意位置点击鼠标：
   - 最新点击点显示为红色实心圆，半径略大便于识别。
   - 过去 3 秒内的其他点击点显示为蓝色实心圆。
3. 超过 3 秒未点击的点会自动淡出并消失，只保留最近的轨迹。

## 代码结构
- `main.cpp`：主窗口实现与鼠标点击逻辑（已添加中文注释，便于阅读）。
- `CMakeLists.txt`：CMake 工程配置，开启了自动 MOC/UIC/RCC，默认使用 C++17。
